# mc-fe

---
提报系统-前端

---
技术栈:

* 整体：webpack1.x + vue1.x
* UI库：iview
* service层：superagent
* 多页面多入口方式开发，暂不使用vue-router
* 其他

---
目录结构:

* src：页面源码
* dist：编译后的js、css等资源文件
* config: webpack等配置文件
* config/VENDORS：公共js/css文件路径

---
项目启动:

* 开发：npm run dev
* 编译：npm run build

---
分支说明:
* 生产环境：master
* 预发环境：beta
* 开发版: dev
* 预发版: devbeta
* 线上版: devmaster
